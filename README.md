# Credit Card Customer Churn Prediction

### Business Goal
Predict the likelihood of customer churn in credit card services to support proactive retention strategy and reduce revenue loss.

### Tools & Methods
Python · Pandas · NumPy · Scikit-learn · Random Forest · Boosting · Model Evaluation

### Approach
- Performed data preprocessing and feature engineering
- Trained tree-based models to classify churners vs non-churners
- Tuned hyperparameters to improve model performance and interpretability

### Result
Improved prediction precision and generated actionable insight for churn reduction strategies.
